Three Basic Principles

The division principle

The home plate umpire takes all calls at home plate and 3rd base and the base umpire takes all calls at 1st and 2nd bases. There are four exceptions:

  • When the batter-runner goes to 3rd base, the base umpire goes with them to make the call at 3rd

  • On infield plays, the base umpire takes the first call on a base, even if it is at 3rd base

  • When a runner steals to 3rd base, the base umpire takes the call

  • When the base umpire is positioned between 2nd and 3rd bases and there is an attempted pick-off by the catcher to 1st base, the plate umpire takes the call

The infield/outfield principle

  • When the ball is in the infield, the base umpire moves or stays in the outfield

The leading runner principle

This principle dictates that the base umpire should start in a position that is up with the leading base runner.

  • With a base runner at 1st base (only), the base umpire will start between 1st and 2nd bases

  • With a base runner at 2nd base, the base umpire will start between 2nd and 3rd bases

  • Note: This allows the base umpire to make calls at the base the leading base runner is advancing to. Also, on a pick-off play, the catcher is more likely to make a play on the leading base runner and the base umpire needs to be in a position to make this call.
